# Autonomous Constellation Manager 🛰️💥

An advanced, high-fidelity orbital mechanics simulator and real-time visualization dashboard designed for the **National Space Hackathon 2026**. This system simulates an autonomous satellite constellation negotiating safe maneuvers inside a dense, high-frequency space debris cloud.

---

## 🚀 Key Features

### 🧠 Advanced Orbital Physics Engine
* **RK4 Integration:** Leverages a 4th-order Runge-Kutta movement engine for ultra-precise state propagation.
* **High-Fidelity Perturbations:** Incorporates Earth's oblateness ($J_2$ perturbation coefficient) rather than just standard spherical gravity.
* **Mass Dynamics:** Calculates precise fuel expenditure using the Tsiolkovsky Rocket Equation.
* **Spatial KD-Tree:** Employs logarithmic $O(\log n)$ proximity searches to reliably detect collision risks across 10,000+ debris elements in real-time.

### 🌐 Dynamic Ground Station Control & Radar
* Real-time line-of-sight (LOS) and elevation angle calculations for globally distributed ground networks (ISTRAC, Svalbard, McMurdo, etc.).
* True day/night mapping generated by calculating the Sun's actual sub-point via Julian dating.
* **Polar Conjunction Radar:** A targeted bullseye tracking interface evaluating real-time threat vectors within critical boundaries.

### 📊 Modern Tech Stack Visuals
* Blazing fast WebGL/2D rendering handled via **PixiJS (v8)**.
* Continuous active-state synchronization via 1-second interval RESTful API polling.
* Real-time dynamic heatmapping tracking visual resource drains and fuel drop-offs.

## 📦 Installation & Setup

### Option 1: Docker Deployment (Recommended for Grading)
To run the system in a completely isolated, unified container matching the evaluation environment:

1. Ensure **Docker Desktop** is installed and running on your system.
2. Open your terminal at the root directory of the project and build the image:
   ```bash
   docker build -t orbital-insight .
3. Run the container and map it to the requested evaluation port:
   ```bash
    docker run -p 8000:8000 orbital-insight
4. Access the full interactive command dashboard at: http://localhost:8000
Note for Windows Users: If your terminal says "docker is not recognized", make sure Docker Desktop is launched. If it still fails, restart your terminal or use Git Bash to run the commands.

### Option 2: Manual Local Setup
Backend Setup:
Clone the project repository and create a virtual environment:

   ```bash
